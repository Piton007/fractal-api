# Fractal Prueba API

## Ver Swagger documentation
 ```javascript
    cd/api
    npm start
    visitar el endpoint /api/docs 
  ```

## Iniciar API /api/
 ```javascript
    cd/api
    npm start 
  ```
## Correr Migraciones
 ```javascript
    cd/api
    npm run migrations <cant> 
  ```
## Iniciar WEB
 ```javascript
    cd/web
    npm run start 
  ```
